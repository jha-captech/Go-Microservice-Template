# Go AWS Lambda Template - SQS Lambda

A mono lambda project scaffold following Go best practices and conventions.

## Instruction

### Setup

Required dependencies:

- Go 1.22+
- Docker
- Docker-Compose
- AWS SAM CLI

### Run

#### SAM Local - create users

```zsh
make lambda_local_create_users
```
